## Печатные формы

Печатные формы используются для автоматизированной подготовки различных служебных документов (сертификатов, протоколов, справок и т.д.), содержащих данные об объекте системы, в одном из печатных форматов — MS Word, MS Excel, PDF или HTML.

Вид формируемого документа и состав данных в нем определяются шаблоном (файл в формате xml или html). Файл шаблона создается предварительно самостоятельно и содержит код в синтаксисе JavaScript, определяющий параметры выходного документа. Файл шаблона привязывается к печатной форме. Печатная форма в свою очередь привязывается к типу объектов базы. По команде "Печать" (из карточки объекта или на Портале) формируется документ указанного в печатной форме формата и содержащий данные, определенные шаблоном.

Печатная форма в зависимости от типа может содержать html разметку и серверный JS код.

Печатную форму с веба можно открыть по ссылке:

```js
var sid = tools_web.get_sum_sid( <ID печатной формы>, Session.sid )

view_print_form.html?print_form_id=<ID печатной формы>&object_id=<ID печатаемого объекта>&sid=<SID-код>
```

Внутри себя в серверном JS в форме определены глобальные объекты curUser, curObject, Request.
